# 00_restructure.md

## Goal
Establish a 3-layer architecture for the Video Converter project.

## Project Structure
- `directives/`: Standard Operating Procedures
- `execution/`: Deterministic scripts
- `src/`: Source code
  - `src/video_converter/`: Main Python package
- `tests/`: Tests
- `.tmp/`: Intermediate files

## Usage
- Run application: `python3 src/video_converter/main.py`
